dom-top-layer playground

This page imports ../dist/index.js and exercises the package with progressively more complex top-layer interactions, including nested flows.

Example 1: Single modal dialog

Start with the simplest case and inspect the emitted enter and exit events.

Example 2: Dialog plus popover

Open a dialog, then toggle a popover from within it to watch the stack grow.

Example 3: Nested top-layer sequence

From the layered dialog, open a nested popover and then a confirmation dialog above everything else.

  • Expected order: layered dialog, nested popover, confirmation dialog.
  • Close them in reverse order to watch the stack unwind.

Example 4: Inject an open dialog

Open a host modal, then inject a new <dialog open> into the page while that modal is still active.

  • This is useful for checking how DOM insertion interacts with top-layer tracking.
  • Expected behavior: the injected dialog is tracked immediately, but while the host modal is open it may sit behind that modal's backdrop.
  • Close the host modal after injection to reveal the injected dialog visually.
